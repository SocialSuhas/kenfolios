Register a WhatsApp Group

The KenFolios Community has announced that members will be allowed to create WhatsApp groups under the community name for cities, interests, and industries starting August 2025. Members can use this form to submit a WhatsApp group to be listed as an official community WhatsApp group.

Currently, we are allowing only free groups. Soon, members will also be able to create and run paid groups for community members. Only paid KenFolios community members will be allowed to join these groups. Upon submitting a request here, we will review and approve it. This process may take 2–3 working days.
• All fields in this form are mandatory
• Only paid KenFolios members can submit requests
• Only paid community members will be allowed to join the groups
• Group name must start with KenFolios
• All submissions are manually reviewed
